# Geturn 컬렉션 메서드{#geturn-collection-method}

이 메서드는 이 컬렉션에 대한 URN를 반환합니다. 이 메서드를 실행하기 전에 Createorupdate () 를 실행해야 합니다.

## Java 예제 {#section_nyl_ycs_rz}

```
collection.getUrn(); 
```

샘플 출력:

```
"urn:livefyre:network=`example.fyre.co`:site=1:collection=1" 
```
