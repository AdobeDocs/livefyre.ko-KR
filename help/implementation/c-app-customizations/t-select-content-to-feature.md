# 앱에서 기능 선택{#select-content-to-feature-from-an-app}

앱에서 직접 기능을 추가하거나 기능 해제

1. 앱 내에서 기능을 사용할 컨텐츠를 가리킵니다.
1. 컨텐츠를 **[!UICONTROL Feature]** 기능하거나 해당 **[!UICONTROL Unfeature]** 콘텐츠의 기능을 중단하려면 클릭하십시오.
