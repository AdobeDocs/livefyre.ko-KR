# 앱{#select-content-to-feature-from-an-app}에서 기능할 콘텐츠 선택

앱에서 바로 콘텐츠를 기능하거나 해제할 수 있습니다.

1. 앱 내에서 기능할 콘텐츠 위로 마우스를 가져갑니다.
1. 컨텐츠의 특징을 지정하려면 **[!UICONTROL Feature]**, 해당 컨텐츠의 특징을 나타내려면 **[!UICONTROL Unfeature]** 을 클릭합니다.
