# 앱에서 기능할 콘텐츠 선택{#select-content-to-feature-from-an-app}

앱에서 바로 콘텐츠를 기능하거나 해제할 수 있습니다.

1. 앱 내에서 기능할 콘텐츠 위로 마우스를 가져갑니다.
1. 컨텐츠를 **[!UICONTROL Feature]** 표시하거나 해당 컨텐츠의 기능을 **[!UICONTROL Unfeature]** 중지하려면 클릭합니다.
