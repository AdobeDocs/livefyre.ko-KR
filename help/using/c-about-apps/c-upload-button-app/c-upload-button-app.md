# 업로드 단추{#upload-button}

업로드 버튼은 사이트 방문자가 컨텐츠를 업로드할 수 있는 방법을 제공합니다.

사이트에서 직접 사용자 생성 콘텐츠를 수집하고 모든 Livefyre 앱에 해당 콘텐츠를 표시할 수 있습니다. 사이트 방문자가 컨텐츠를 업로드함으로써 대화에 기여할 수 있도록 유도합니다. Livefyre는 이미지, 비디오 및 오디오 컨텐츠를 지원합니다.
