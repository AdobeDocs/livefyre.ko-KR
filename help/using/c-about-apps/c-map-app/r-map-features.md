# 맵 기능{#map-features}

맵 앱은 사이트에 대해 이러한 기능을 제공합니다.



* 컨텐츠가 실시간으로 맵에 들어갑니다.
* 동일한 영역의 컨텐츠는 클러스터되며 갤러리 모달에서 볼 수 있습니다.
* 맵으로 확대하면 특정 영역의 컨텐츠에 집중할 수 있습니다.
