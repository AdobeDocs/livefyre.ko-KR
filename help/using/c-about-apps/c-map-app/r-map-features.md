# 지도 기능{#map-features}

맵 앱은 사이트에 대해 이러한 기능을 제공합니다.



* 컨텐츠는 실시간으로 맵에 입력됩니다.
* 동일한 영역의 컨텐츠가 클러스터되어 갤러리 모달에서 볼 수 있습니다.
* 맵을 확대하여 특정 영역의 컨텐츠에 초점을 맞춥니다.
