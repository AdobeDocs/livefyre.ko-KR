# 투표{#polls}

고객이 실시간으로 의견을 제시하도록 할 수 있습니다.

투표를 통해 한 번의 질문에 대한 빠른 투표를 사이트에 만들고 게시할 수 있습니다. 투표를 사용하여 Reader 기여도를 높이고, 사이트 방문자의 피드백, 의견 및 요청을 수집합니다.

>[!NOTE]
>
>투표는 스트림에 연결되지 않습니다.
