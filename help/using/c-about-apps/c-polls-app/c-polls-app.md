# 투표{#polls}

고객이 실시간으로 의견을 전달할 수 있습니다.

투표를 사용하면 사이트에 1개의 질문이 있는 빠른 투표를 만들어 게시할 수 있습니다. 투표를 사용하여 방문자의 참여를 유도하고, 피드백, 의견 및 사이트 방문자의 요청을 수집합니다.

>[!NOTE]
>
>투표는 스트림에 연결되지 않습니다.
