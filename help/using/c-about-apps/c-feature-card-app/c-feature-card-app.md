# 기능 카드{#feature-card}

페이지에 주요 콘텐츠 한 개 추가

기능 카드를 사용하면 하나의 카드 디스플레이에서 개별 컨텐츠를 강조 표시할 수 있습니다. 앱은 최신 주요 콘텐츠를 우선 배치하고 최신 항목을 순환하여 최신 콘텐츠와 연관성 높은 디스플레이를 보장합니다.

Feature Card는 가장 최근에 게시된 항목(반드시 가장 최근에 게시된 항목 아님)을 표시하므로 페이지에 적시에 표시할 최상의 컨텐츠를 선택할 수 있습니다.
