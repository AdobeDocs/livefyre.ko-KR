# 사이트에 번역 세트 적용{#apply-a-translation-set-to-a-site}

**사이트에 번역 세트 적용**

1. 스튜디오 설정으로 이동합니다.
1. 사이트를 선택합니다.
1. 사이트 설정을 선택합니다.
1. 앱 설정을 선택합니다.
1. 기본 설정 > 번역 세트로 이동합니다.
1. 드롭다운 메뉴에서 번역 세트를 선택합니다.
