# 번역 세트를 사이트{#apply-a-translation-set-to-a-site}에 적용

**사이트에 번역 세트 적용**

1. Studio의 설정으로 이동합니다.
1. 사이트를 선택합니다.
1. 사이트 설정을 선택합니다.
1. 앱 설정을 선택합니다.
1. 기본 설정 > 번역 세트로 이동합니다.
1. 드롭다운 메뉴에서 번역 세트를 선택합니다.
