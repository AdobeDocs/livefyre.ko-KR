# 네트워크에 번역 세트 적용{#apply-a-translation-set-to-a-network}

1. 스튜디오 설정으로 이동합니다. 사이트 수준이 아닌 네트워크 수준으로 작업하고 있는지 확인합니다.
1. [네트워크 설정] 를 선택합니다.
1. 앱 설정을 선택합니다.
1. 기본 설정 > 번역 세트로 이동합니다.
1. 드롭다운 메뉴에서 번역 세트를 선택합니다.
