# 개인 정보 보고서 보기{#view-a-privacy-report}

Livefyre의 개인정보 보호 보고서를 통해 사용자를 위해 Livefyre에 기록되는 정보를 확인할 수 있습니다.

1. 이동 **[!UICONTROL Settings > Privacy]**.
1. 목록에서 개인 정보 보고서를 클릭하여 표시합니다.
