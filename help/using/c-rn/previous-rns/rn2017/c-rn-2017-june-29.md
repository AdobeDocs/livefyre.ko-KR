## 프로덕션 릴리스

| **문제 유형** | **구성 요소** | **릴리스 노트** |
|---|---|---|
| 버그 | 라이브러리 | 라이브러리에서 필터를 지우면 다른 페이지로 리디렉션되는 대신 동일한 폴더 선택 위치에 유지됩니다. |
